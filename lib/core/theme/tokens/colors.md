# Color Tokens

## Brand Colors
Based on MAAWA logo gradients with AA contrast compliance.

### Primary Palette
- **Primary Coral**: `#FF6B6B` - Main brand color from logo gradient
- **Primary Magenta**: `#E91E63` - Secondary brand color from logo gradient  
- **Primary Turquoise**: `#26C6DA` - Accent color from logo
- **Primary Orange**: `#FF8A65` - Supporting brand color

### Light Theme Colors
- **Background**: `#FAFAFA` - Main app background
- **Surface**: `#FFFFFF` - Card and component backgrounds
- **On Surface**: `#1A1A1A` - Primary text on surfaces
- **On Background**: `#2C2C2C` - Primary text on backgrounds
- **Secondary**: `#757575` - Secondary text and icons
- **Tertiary**: `#9E9E9E` - Disabled text and subtle elements

### Dark Theme Colors
- **Background**: `#121212` - Main app background
- **Surface**: `#1E1E1E` - Card and component backgrounds
- **On Surface**: `#E0E0E0` - Primary text on surfaces
- **On Background**: `#FFFFFF` - Primary text on backgrounds
- **Secondary**: `#B0B0B0` - Secondary text and icons
- **Tertiary**: `#808080` - Disabled text and subtle elements

### Semantic Colors
- **Success**: `#4CAF50` - Success states, confirmations
- **Warning**: `#FF9800` - Warnings, cautions
- **Error**: `#E53935` - Errors, destructive actions
- **Info**: `#2196F3` - Information, neutral notifications

### Neutral Grays
- **Gray 50**: `#FAFAFA`
- **Gray 100**: `#F5F5F5`
- **Gray 200**: `#EEEEEE`
- **Gray 300**: `#E0E0E0`
- **Gray 400**: `#BDBDBD`
- **Gray 500**: `#9E9E9E`
- **Gray 600**: `#757575`
- **Gray 700**: `#616161`
- **Gray 800**: `#424242`
- **Gray 900**: `#212121`

### State Variants
Each color includes hover, pressed, and disabled variants:
- **Hover**: -10% lightness
- **Pressed**: -20% lightness  
- **Disabled**: 40% opacity

### Shadow Colors
- **Light**: `rgba(0, 0, 0, 0.1)` - Subtle shadows
- **Medium**: `rgba(0, 0, 0, 0.2)` - Standard shadows
- **Dark**: `rgba(0, 0, 0, 0.3)` - Prominent shadows

### Gradients
- **Primary Gradient**: Linear from Primary Orange → Primary Coral → Primary Magenta
- **Accent Gradient**: Linear from Primary Turquoise → Cyan
