# Typography Tokens

## Font Families

### English Typography
- **Primary**: Google Fonts - Inter
- **Secondary**: Google Fonts - Poppins (for headings)
- **Fallback**: System fonts (Roboto on Android, SF Pro on iOS)

### Arabic Typography  
- **Primary**: Google Fonts - Almarai
- **Secondary**: Google Fonts - Cairo (for headings)
- **Fallback**: System Arabic fonts

## Type Scale

### Display Styles
- **Display Large**: 32px / 40px line height, Bold (700)
- **Display Medium**: 28px / 36px line height, Bold (700)  
- **Display Small**: 24px / 32px line height, SemiBold (600)

### Headline Styles
- **Headline Large**: 22px / 28px line height, SemiBold (600)
- **Headline Medium**: 20px / 26px line height, SemiBold (600)
- **Headline Small**: 18px / 24px line height, SemiBold (600)

### Title Styles
- **Title Large**: 16px / 22px line height, SemiBold (600)
- **Title Medium**: 14px / 20px line height, Medium (500)
- **Title Small**: 12px / 18px line height, Medium (500)

### Body Styles
- **Body Large**: 16px / 24px line height, Regular (400)
- **Body Medium**: 14px / 20px line height, Regular (400)
- **Body Small**: 12px / 18px line height, Regular (400)

### Label Styles
- **Label Large**: 14px / 20px line height, Medium (500)
- **Label Medium**: 12px / 18px line height, Medium (500)
- **Label Small**: 10px / 16px line height, Medium (500)

## Font Weights
- **Light**: 300
- **Regular**: 400
- **Medium**: 500
- **SemiBold**: 600
- **Bold**: 700

## Letter Spacing
- **Tight**: -0.5px (for large headings)
- **Normal**: 0px (default)
- **Wide**: 0.5px (for small labels)

## RTL Considerations
- Arabic text uses right-to-left reading direction
- Line height adjusted +2px for Arabic fonts
- Letter spacing disabled for Arabic text
- Text alignment automatically mirrors in RTL mode
