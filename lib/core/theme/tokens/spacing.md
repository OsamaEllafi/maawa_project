# Spacing Tokens

## Base Unit
All spacing follows a 4pt grid system for consistency and alignment.

## Spacing Scale
- **xs**: 4px - Minimal spacing, tight layouts
- **sm**: 8px - Small spacing, compact elements  
- **md**: 12px - Medium spacing, standard gaps
- **lg**: 16px - Large spacing, section breaks
- **xl**: 20px - Extra large spacing, major sections
- **2xl**: 24px - Double extra large, page margins
- **3xl**: 32px - Triple extra large, hero sections
- **4xl**: 40px - Quad extra large, major breaks
- **5xl**: 48px - Quint extra large, page sections
- **6xl**: 64px - Sext extra large, major layouts

## Component Spacing

### Padding
- **Button**: 12px horizontal, 8px vertical (sm/xs)
- **Card**: 16px all sides (lg)
- **Input**: 16px horizontal, 12px vertical (lg/md)
- **List Item**: 16px horizontal, 12px vertical (lg/md)

### Margins
- **Section**: 24px bottom (2xl)
- **Component**: 16px bottom (lg)
- **Element**: 8px bottom (sm)

### Gaps
- **Grid**: 16px (lg)
- **Flex**: 8px (sm)
- **Stack**: 4px (xs)

## Border Radius
- **xs**: 2px - Small elements, badges
- **sm**: 4px - Buttons, chips
- **md**: 8px - Cards, inputs
- **lg**: 12px - Large cards, modals
- **xl**: 16px - Hero cards, major components
- **2xl**: 20px - Special components
- **full**: 9999px - Pills, circular buttons

## Elevation/Shadow
- **0**: No shadow
- **1**: 0px 1px 2px rgba(0,0,0,0.1) - Subtle lift
- **2**: 0px 2px 4px rgba(0,0,0,0.1) - Standard cards
- **3**: 0px 4px 8px rgba(0,0,0,0.15) - Elevated cards
- **4**: 0px 8px 16px rgba(0,0,0,0.15) - Floating elements
- **5**: 0px 16px 32px rgba(0,0,0,0.2) - Modals, overlays
